데이터 분석 방법론

회귀분석

1. 관심 공변량을 설정 ( 우리가 생각했을때, 영향을 반드시 줄 것이라고 예상되는 변수 설정)

2. 관심 공변량을 바탕으로 탐색적자료분석(EDA)

3. EDA 결과를 바탕으로 어떤 회귀모형을 설정할 것인지 설계
- 반응변수가 위험지역인지 아닌지 여부를 생각할 때 -> 로지스틱 회귀모형
- 반응변수가 사고다발지역인지 생각할 때(예상 사고가 일어난 횟수) -> 포아송 회귀모형
ㄴ 여기서 가장 큰 문제가 관심 공변량에 따른 반응변수가 우리가 흔히 아는 분포를
따르지 않을 때, 비모수적방법으로 해결해야되는데 나도 어려움

4. 회귀모형 가정성 설명
- 선형성, 독립성, 정규성, 등분산성 (L I N E 가정 충분히 설명)

5. 모형화 (Modeling)
- 모형화 후 유의한 변수들을 판단한다. (유의수준보다 작은 유의확률을 가지는 변수들)
- EDA 결과와 모형화 최종 결과에 따라 회귀 모형식으로 표현
- odds ratio, 회귀계수 등등으로 변수들의 자세한 특징들 설명

6. 문제점
- 적합결여검정을 통해 모델링이 잘 설명이 되었는지? 확인해보기
- 이상치, 과대산포, 다중 공선성 등 모형화에 이상을 주는 요소들을 파악 후 다시 모델링

7. 결론
- 최종모형을 바탕으로 변수 선택

머신러닝
1. 데이터셋 구성
- 회귀분석 및 탐색적자료분석 결과를 바탕으로 Training set, Test set 나누기 (비율)

2. 사용할 ML 방법론은? (방법론에 대한 설명은 더 공부해서 알려드리겠습니다. 저도 실전에서 사용하는건 처음입니다..)
ㄱ. Multi Layer Perception (MLP)

ㄴ. Support Vector Machine (SVM)

ㄷ. Decision Tree (DT, 의사결정나무)

ㄹ. Random Forest(RF)

ㅁ. Naive Bayes Classification(NBC, 나이브 베이즈 분류기)

3. 성능지표 확인하기

ㄱ. Confusion Matrix

ㄴ. MSE (보통 논문에선 RMSE를 주로 채택)

ㄷ. AIC 또는 BIC 로 성능 비교
